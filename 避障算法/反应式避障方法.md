# 反应式避障方法
反应式避障方法顾名思义，就是不是使用规划式的避障方法，而是使用对障碍物的临时反应进行避障。反应式方法是一种实时的、基于传感器输入的避障策略。机器人在感知到障碍物后，立即做出反应以避免碰撞。这些方法通常基于简单的规则或算法，例如避免障碍物的最短距离或避免与其他物体的碰撞。这种臂章方法相对于规划式的避障方法，响应速度与效率更高。并且反应式的避障方法一般都是用在局部的环境中，搜索的范围更小，延迟更加低。更适合机器人在室内外的这种需要高效避障的场景。
# 反应式避障
反应式避障方法目前看到有几篇，其中ETH在他们的多分辨率三维地图，使用RMP(基于黎曼几何进行约束规划)的方法，提出了一种在无人机上实现的反应式避障方法。由于RPM是一种使用对加速度进行规划的方法，所以在位置上天然的具有平滑性，并且能够高效的进行计算。参考文献在[Waverider: Leveraging Hierarchical, Multi-Resolution Maps  for Efficient and Reactive Obstacle Avoidance]
第二篇是一种通过计算地图上深度差，获得当前地图是否有机器人通过的gap的方法，参考论文在[https://github.com/mazrk7/reactive_assistance],由于这个代码的原理较为简单，因此目前主要在学习这个代码的原理。
